# atividade-1-flask-crud-Pietro-Pedroh
A atividade consiste na criação de um sistema de cadastro de peças em uma Auto-Peças.
Onde nesse sistema sera permitido:
  * Cadastrar novas peças(Tendo como atributos: Nome, descrição, veiculos compativeis e preço)
  * Consultar peças já cadastradas
  * Editar informações de peças já cadastradas
  * Excluir o cadastro das peças

Tecnologias utilizadas:
  * Python 3
  * Flask
  * HTML5 CSS3
  * Jinja2
